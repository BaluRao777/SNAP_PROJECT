# Dataset (Kaggle Facial Keypoints)

| File | Description |
|------|-------------|
| `training.csv` | Labeled training set (~7k faces, 96×96 pixels + 30 landmark columns) |
| `test.csv` | Test images without labels |
| `IdLookupTable.csv` | Competition lookup table |
| `SampleSubmission.csv` | Submission format template |

Download from [Kaggle — Facial Keypoints Detection](https://www.kaggle.com/c/facial-keypoints-detection/data) if these files are missing.
